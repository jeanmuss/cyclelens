# Product Log

## 当前功能总览

- 加密周期地图：BTC、ETH、SOL、HYPE 月度收益矩阵、相对 BTC 收益、四年周期分组、读图结论、月内高低点顺序、固定详情卡和 URL 状态同步。
- 美股宏观：QQQ、SPY、DIA 代理价格与 FRED 宏观数据的周度观察页，支持移动端 2x2 摘要、周度表格和可分享 hash 状态。
- 事件与流动性：通胀、就业增长、利率美元、波动信用、流动性和人工流动性事件日历，周历/月历会随本地日期自动定位。
- 开市轮动：加密、美股、韩股和 A 股等市场的开闭市状态、样例行情、来源质量和轮动列表，移动端摘要改为 2x2。
- AI 芯片产业链：用细分类目和股票代码观察晶圆设备、材料基板、制造 IDM、存储 HBM、AI 芯片 IP、光模块互联、服务器、数据中心基础设施、软件终端应用和其他云算力需求的轮动。
- 机器人链：复刻机器人产业链观察池，按算力、感知、芯片、运动控制、自动驾驶、仓储服务机器人、医疗机器人、防务无人系统和 ETF 展示价格、涨幅和简要 K 线。
- 行情与缓存：前端只读静态 JSON；加密、市场时钟、美股宏观、事件流动性、AI 芯片链和机器人链分别由本地/CI 脚本生成缓存，密钥不进入浏览器。

## 2026-06-30

- 流动性页面新增“卖压渐竭”字段：每月最后一个周一到周五自动显示，并使用独立浅橙底色区分月末/季末窗口。
- AI 产业链确认并展示 AMAT：AMAT 位于晶圆设备类目，与 ASML、LRCX 共同构成设备观察项。
- AI 产业链和机器人链共用真实 K 线数据入口：`update-chip-chain-data.mjs` 支持 `chip` / `robot` 两个目标，有 Alpaca 凭证时写入真实 `pricePaths`，无凭证时保留 last-known-good。
- GitHub Actions 增加机器人链缓存刷新步骤：Pages 部署和低频市场缓存工作流都会尝试刷新 `robot-chain-watchlist.json`。
- 新增仓库根目录 `README.md`，补齐 GitHub 首页缺失的项目说明、数据原则、开发命令、Secrets 和部署说明。
